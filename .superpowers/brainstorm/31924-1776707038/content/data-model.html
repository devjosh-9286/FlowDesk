Key Design Decisions
• Flow nodes stored as JSON in FlowTemplate — flexible, no rigid node tables, queryable via Postgres JSONB
• Project clones snapshot of template — template changes don't affect running projects
• orgId on every row — enforced at API middleware layer, guarantees tenant isolation
• Conditions as JSON rules — e.g. {"field":"priority","op":"eq","value":"high","goto":"nodeId"}
